Setup TX (Dongle)

* Open nRFgo studio
* Program Softdevice
  
  * Use s120_nrf51_2.1.0_softdevice.hex in this folder

* Program application

  * .\ble_central_app_uart\pca10031\s120\arm5\_build\nrf51422_xxac_s120.hex


Setup RX (beacon board)

* Open nRFgo studio
* Program Softdevice
  
  * Use s110_nrf51_8.0.0_softdevice.hex in this folder

* Program application

  * .\ble_peripheral_app_uart\pca20006\s110\arm5\_build\nrf51422_xxac_s110.hex
